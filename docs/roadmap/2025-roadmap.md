## Risks & Mitigation

### Risk 1: Scope Creep
**Impact:** High
**Probability:** Medium
**Mitigation:**
- Stick to quarterly themes
- Review backlog monthly, defer non-essential features
- User test early and often to validate priorities

### Risk 2: Technical Debt
**Impact:** Medium
**Probability:** Medium
**Mitigation:**
- Allocate 20% of each sprint to refactoring
- Code reviews for all features
- Keep dependencies updated

### Risk 3: Persona Misalignment
**Impact:** High
**Probability:** Low
**Mitigation:**
- Re-test with personas quarterly
- Track feature adoption per persona type
- Survey users to validate persona accuracy

### Risk 4: Mobile App Delays
**Impact:** Medium
**Probability:** Medium
**Mitigation:**
- Start React Native setup in Q2 (early)
- Hire mobile developer if needed
- De-scope Q4 features if mobile is delayed

### Risk 5: AI Costs
**Impact:** Medium
**Probability:** Low
**Mitigation:**
- Monitor Claude API usage per user
- Cache common queries
- Rate limit Ask AI (reasonable limits)
- Consider premium tier for heavy users
